> nodejs async/await 与gulp的配置组合

## 使用方法

1、克隆代码
```
git clone https://github.com/calidion/async-await-nodejs.git
```
2、安装npm包
```
npm install
# 或者
yarn install
```
3、执行gulp
```
gulp
```
完成相应的测试

4、如果要包做成发布包，修改相应的包名，地址名。
